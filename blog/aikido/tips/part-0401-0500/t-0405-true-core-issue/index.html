合氣道進步的秘訣【第0405回】　什麼才是真正的問題

合氣道是透過相對稽古、在練技過程中逐步精進的修行方式，但要如願上達並不容易。若像其他武道或運動一樣，有勝負可分的比賽，目標會更明確，也容易判斷自己是否進步。然而，合氣道沒有與他人爭勝的對抗，因此判斷起來就困難得多。

在修行初期，或許還能明顯感受到自己的進步。這個階段的稽古，是為了能「把對手摔倒」的相對稽古。從與道場夥伴、前輩的練習中，能感覺出自己的技是否奏效，有時覺得「啊，技終於能用了」，有時又覺得「完全沒進步」。

但隨著時間推進，相對稽古終會轉變為「絕對的稽古」。這時，練習的對象已不再是他人，而是自己。真正的修行，從這裡才算開始。

然而，這之後的稽古與之前完全不同，也更加艱難。若有導師在旁指引固然幸運，但往往到了這個階段，周圍已無人能再教你。此時必須靠自己摸索、靠自己進步。

不過，即使沒人能教導，想要上達依然有其法則可循。從這裡開始，就只能相信那些「上達的法則」並實踐之。

法則可能有許多種，也會因人而異，但那些法則往往不僅適用於合氣道，還通用於人生的其他領域。

曾在電視上看到一位成功的日本創業家談到：在創業之前，他曾前往孕育無數創新事業的美國矽谷，詢問許多成功的創業者：「你是怎麼想到你的事業點子的？」結果，多數人回答說：「最重要的是要知道，你想解決什麼問題。先找出你周遭真正困擾你的事，再用你的技術去解決它。這樣自然就會成為一門事業。」

若把這個原則套用到合氣道上，無論是想讓合氣道成為事業，或是想作為合氣道家精進，這條法則都十分有用。

觀察那些初學者與為上達所困的稽古人，最根本的問題往往在於，他們不知道「問題是什麼」。不是沒有發現問題，就是根本沒去在意。若連問題都不知道，那自然也就無從上達。

一旦察覺問題，就要著手解決。解決的方法，可能是運用自己已有的技術與方法，也可能要創出新的技與術。能夠發現問題、再去解決問題，這本身就是上達的過程。

隨著修行深入，你會發現，問題永遠存在。解決了一個，又會出現下一個。這不斷重複，正是合氣道修行者的宿命。也正因如此，修行才沒有終點，進步也沒有極限。我們能不斷把目標提升到更高處，挑戰不息，這正是合氣道的可貴之處。感謝合氣道。
